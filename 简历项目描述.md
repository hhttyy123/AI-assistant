# AI语音对话助手 - 简历项目描述


## 项目概述

基于 Electron 开发的跨平台智能语音聊天桌面应用，集成 Whisper 语音识别与 GLM-4.6 大模型，实现端到端的语音交互体验。


---

## 版本一：标准版（推荐）

### AI语音对话助手 | 独立开发

基于 Electron 框架开发跨平台桌面应用，集成 Whisper 语音识别与 GLM-4.6 大模型，实现智能语音交互。

**核心技术亮点：**
- 实时语音采集与识别（准确率 95%+）
- 对接 GLM-4.6 API，实现流畅的多轮对话
- TTS 语音合成，支持 AI 回复语音播报
- 双模态交互（语音+文本），聊天记录本地持久化
- 支持 Windows/macOS/Linux 跨平台打包

**技术栈：** Electron、Node.js、Python/Whisper、GLM API、Web Audio API

## 面试话术参考

Q: 请介绍一下这个项目？

A: 这是一个基于 Electron 的智能语音对话助手。我独立完成了整个项目的开发，包括前端界面、语音采集、后端的 Whisper 语音识别集成，以及对接 GLM-4.6 大模型实现 AI 对话。用户可以通过语音或文字与 AI 进行交流，AI 会以语音和文字双重形式回复。项目支持 Windows、macOS 和 Linux 三大平台。

Q: 技术难点是什么？

A: 主要难点有三个：一是实时音频采集与处理，需要使用 Web Audio API 并处理音频格式转换；二是 Python Whisper 模型的集成，需要通过子进程调用并处理数据传输；三是跨平台打包的兼容性问题。最终都通过技术方案解决了。

Q: 有什么创新点？

A: 实现了语音和文本的双模态交互，用户可以随时切换输入方式；同时集成了完整的语音链路（采集-识别-对话-播报），体验流畅自然。
